# Daily-schedule-Manager
A Python program built with Tkinter that displays your daily schedule in a clean GUI window.

Description
A simple Python GUI program using Tkinter that shows your daily tasks and times in a structured table. Tasks are managed directly in the code, and the program displays them in a clean, window. 

Features
- Displays daily tasks in a Time & Task table
- Clean and simple GUI built with Tkinter
- Tasks and times are stored in a dictionary in the code
- Scrollable task list

 Requirements
- Python 3.x
- Tkinter (built into Python)

How To Run
1. Clone the repository
   git clone https://github.com/Mandysvg/daily-schedule-manager.git

2. Navigate to the folder
   cd daily-schedule-manager

3. Run the app
   python schedule.py

 How To Add Tasks
Tasks are added directly in the code. Open the file and edit the dictionary:

   tasks = {
       "08:00": "Wake up",
       "09:00": "Eat breakfast",
       "10:00": "Go to the pool",
       "11:00": "Go to the store"
         (Add here)
   }
